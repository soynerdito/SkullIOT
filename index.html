Skull IOT

Skull Controlled via web

Download this project as a .zip file Download this project as a tar.gz file

About

This is a small weekend project to connect a regular decoration skull to the web. It is a regular skull ornament with a few leds, an Atmega328 and a CC300 wifi module. Using the MQTT protocol it receives messages from the web and turn on leds accordingly.
